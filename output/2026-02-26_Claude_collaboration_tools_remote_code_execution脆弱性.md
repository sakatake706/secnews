Claude collaboration tools remote code execution脆弱性

- **概要**
Claude Codeのセキュリティ脆弱性により、攻撃者がリポジトリ内の設定ファイルを悪用しリモートでコードを実行可能となる問題が発見された。これによりAPIキーの窃取も可能であり、複数の脆弱性が修正されたがAIツールの新たな攻撃面を示す事例である。

- **URL**
https://go.theregister.com/feed/www.theregister.com/2026/02/26/clade_code_cves/

- **備考**
[[Claude Code]]の設計上の欠陥により[[リモートコード実行]]が可能となった。攻撃者はリポジトリの設定ファイルを改変し、他の開発者の環境で任意のコマンドを実行できる。これにはHooks機能の悪用やMCP同意回避が含まれ、APIキーの盗難も確認された。対策として設定ファイルの管理強化やアクセス制御が必要である。攻撃はサプライチェーンリスクを増大させる。
