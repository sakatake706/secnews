- **タイトル**
Windows Task Host脆弱性警告

- **概要**
CISAはWindows Task Hostの特権昇格脆弱性が攻撃に悪用されていると警告した。この脆弱性はWindows 11およびWindows Server 2025に影響し、攻撃者が低権限からSYSTEM権限を取得可能である。Microsoftは2025年11月に修正パッチを公開し、CISAは連邦機関に速やかな対策を求めている。

- **URL**
https://www.bleepingcomputer.com/news/security/cisa-flags-windows-task-host-vulnerability-as-exploited-in-attacks/

- **備考**
この脆弱性は[[CVE-2025-60710]]に該当し、攻撃者はリンク解決の不備を利用して特権を昇格させる。推奨対策はMicrosoftのパッチ適用であり、連邦機関はBOD 22-01に従い迅速な対応が必要である。将来的にも類似の特権昇格攻撃が増加する可能性が高い。
