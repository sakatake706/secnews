- **タイトル**
React Native CLIやSmarterMail脆弱性警戒

- **概要**
米当局は「SmarterMail」や「React Native CLI」の既知脆弱性が悪用されているとして注意喚起を行った。SmarterMailのAPI認証欠如による脆弱性やReact Native CLIのOSコマンドインジェクション脆弱性が確認されている。これらはランサムウェア攻撃で悪用されており、利用者は警戒が必要である。

- **URL**
https://www.security-next.com/180672

- **備考**
「SmarterMail」のAPI認証欠如脆弱性「CVE-2026-24423」や「React Native CLI」のOSコマンドインジェクション脆弱性「CVE-2025-11953」は攻撃者が遠隔からコマンド実行可能とする。ランサムウェア攻撃での悪用が確認されており、速やかなパッチ適用と監視強化が推奨される。類似の脆弱性悪用事例も多く、被害拡大の可能性が高い。
