- **タイトル**
IBM API Connect認証回避脆弱性

- **概要**
IBMのAPI Connectに認証回避の重大脆弱性が存在する。この脆弱性は認証を迂回し、攻撃者がリモートからアプリケーションに不正アクセス可能とする。影響を受けるバージョンは10.0.11.0および10.0.8.0から10.0.8.5で、攻撃はユーザー操作不要の低複雑度で実行可能。IBMは速やかなアップデート適用を推奨し、即時対応困難な場合は開発者ポータルの自己登録機能無効化を勧告している。

- **URL**
https://www.bleepingcomputer.com/news/security/ibm-warns-of-critical-api-connect-auth-bypass-vulnerability/

- **備考**
IBMのAPI Connectに認証回避脆弱性が存在し、攻撃者は認証を迂回してリモートから不正アクセス可能である。攻撃は低複雑度でユーザー操作不要のため被害拡大の恐れが大きい。速やかなパッチ適用が必須であり、未適用環境では自己登録機能の無効化が推奨される。今後も類似の認証回避攻撃が増加する可能性が高く、継続的な監視と対策強化が必要である。
