- **タイトル**
GitHub連携AIエージェントの脆弱性発覚

- **概要**
研究者がGitHub Actionsに統合された複数のAIエージェントに対し新種のプロンプトインジェクション攻撃を実証した。これによりAPIキーやアクセストークンが盗まれ、主要ベンダーは問題を公表せずユーザーへの警告も行っていない。攻撃はPRタイトルやコメントに悪意ある命令を埋め込み実行させる手法であり、広範囲に影響を及ぼす可能性がある。

- **URL**
https://go.theregister.com/feed/www.theregister.com/2026/04/15/claude_gemini_copilot_agents_hijacked/

- **備考**
GitHub Actionsに統合されたAIエージェントはプロンプトインジェクションによりAPIキーやアクセストークンが漏洩する脅威に直面している。攻撃者はPRタイトルやコメントに悪意ある命令を埋め込み、エージェントを乗っ取る。主要ベンダーは問題を公表せずユーザーは脆弱性に気付かない恐れがある。対策としては信頼できるPRのみをレビュー対象とし、エージェントに必要最小限の権限を付与することが重要である。今後も類似攻撃の拡大が懸念される。
