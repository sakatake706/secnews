- **タイトル**
CISA警告CentOS Web Panel脆弱性悪用

- **概要**
米国のサイバーセキュリティ機関CISAはCentOS Web Panelに存在する重大なリモートコマンド実行脆弱性が攻撃に悪用されていると警告した。この脆弱性は認証なしで有効なユーザー名を知る攻撃者が任意のシェルコマンドを実行可能とするものである。影響を受けるバージョンは0.9.8.1204以前で、修正済みの0.9.8.1205がリリース済みである。連邦機関には11月25日までに対策実施が求められている。

- **URL**
https://www.bleepingcomputer.com/news/security/cisa-warns-of-critical-centos-web-panel-bug-exploited-in-attacks/

- **備考**
[[CVE-2025-48703]]の脆弱性は認証不要のコマンド実行を許すため攻撃者は遠隔からシェルを取得可能である。攻撃はファイルマネージャのchangePermエンドポイントの不適切なパラメータ処理に起因し、迅速なパッチ適用が推奨される。類似の脆弱性は過去にも悪用例があり、監視強化と早期対応が重要である。攻撃者[[不明]]の活動継続が懸念される。
