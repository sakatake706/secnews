- **タイトル**
AppsFlyer Web SDK乗っ取り

- **概要**
AppsFlyerのWeb SDKが一時的に悪意あるコードにより乗っ取られ、暗号通貨を盗むJavaScriptコードが拡散された。攻撃者はウォレットアドレスを検知し、自身のアドレスに書き換えて資金を横取りした。SDKは多くのアプリで利用されており影響範囲は広範囲に及ぶ。攻撃は2026年3月9日から11日までの間に発生し、調査は継続中である。

- **URL**
https://www.bleepingcomputer.com/news/security/appsflyer-web-sdk-used-to-spread-crypto-stealer-javascript-code/

- **備考**
攻撃者は[[JavaScriptコード]]を用いウォレットアドレスを乗っ取り資金を奪取する。SDKの信頼性を悪用し多くのアプリに影響を与えた。対策としてSDKのバージョン確認と疑わしいAPI通信の監視が必要である。将来的に類似のサプライチェーン攻撃が増加する可能性が高い。
