- **タイトル**
Critical vm2 sandbox bug allows code execution

- **概要**
人気のNode.jsサンドボックスライブラリvm2に重大な脆弱性が発見された。この脆弱性はCVE-2026-26956として追跡されており、特にNode.js 25環境でWebAssembly例外処理が有効な場合に影響を及ぼす。攻撃者は特殊なTypeErrorを利用しホスト側のエラーオブジェクトをサンドボックス内に漏洩させ、Node.jsの内部にアクセスして任意のコードを実行可能となる。利用者は速やかにバージョン3.10.5以降へ更新すべきである。

- **URL**
https://www.bleepingcomputer.com/news/security/critical-vm2-sandbox-bug-lets-attackers-execute-code-on-hosts/

- **備考**
vm2の脆弱性は例外処理の誤りに起因し、攻撃者は[[CVE-2026-26956]]を悪用してホスト上で任意コードを実行可能である。対策として最新版への更新が推奨され、類似の過去脆弱性も存在するため継続的な監視が必要だ。今後もサンドボックス回避の手法が進化する可能性が高く、早期対応が被害軽減に重要となる。
