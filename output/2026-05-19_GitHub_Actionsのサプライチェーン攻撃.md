- **タイトル**
GitHub Actionsのサプライチェーン攻撃

- **概要**
脅威アクターがGitHub Actionsのactions-cool/issues-helperを改ざんし悪意あるコードを実行して機密情報を窃取し攻撃者のサーバーに送信している。既存のタグは偽のコミットに移動され正常な履歴に含まれない。

- **URL**
https://thehackernews.com/2026/05/github-actions-supply-chain-attack.html

- **備考**
この攻撃はソフトウェア供給網の脆弱性を突いており開発者は依存関係の監査と権限管理を強化すべきである。継続的監視とアクセス制御の徹底が推奨される。
