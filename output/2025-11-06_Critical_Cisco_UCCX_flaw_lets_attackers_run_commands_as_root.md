- **タイトル**
Critical Cisco UCCX flaw lets attackers run commands as root

- **概要**
Cisco Unified Contact Center ExpressのJava RMIプロセスに存在する重大な脆弱性により、認証されていない攻撃者がリモートで任意のコマンドをroot権限で実行可能である。この脆弱性は不適切な認証機構に起因し、悪用されるとシステムの完全制御を許す恐れがある。Ciscoは複数の関連製品に対し緊急のセキュリティアップデートを提供した。

- **URL**
https://www.bleepingcomputer.com/news/security/critical-cisco-uccx-flaw-lets-hackers-run-commands-as-root/

- **備考**
この[[脆弱性]]は攻撃者がroot権限を取得しシステム制御を奪うことを目的とし、Java RMIプロセスを悪用する手法が確認されている。推奨対策は速やかなアップデート適用であり、類似の過去事例から被害拡大の可能性が高いと判断される。今後も同種の攻撃が増加する恐れがあるため監視強化が必要である。
