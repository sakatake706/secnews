- **タイトル**
Device Code Phishing急増

- **概要**
複数の高度なフィッシングキットがデバイスコードを狙いパスワードなしでアカウント侵害を行う手法を利用している。これによりMicrosoft365のアクセストークンを不正取得し持続的アクセスを確保する攻撃が増加している。主要プラットフォームは機能を強化し攻撃の高度化が進んでいる。今後も防御回避技術の進化が予想される。

- **URL**
https://blog.knowbe4.com/device-code-phishing-tsunami

- **備考**
デバイスコードフィッシングは正規認証を悪用しMicrosoft365のトークンを不正取得する攻撃である。攻撃者はこれを用いて持続的アクセスを維持し追加攻撃を展開する。主要キットはAI技術を統合し機能を強化しており被害拡大の可能性が高い。対策として多要素認証強化や異常検知の導入が推奨される。今後も攻撃手法の高度化に注意が必要である。
