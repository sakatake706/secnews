- **タイトル**
Exchangeゼロデイ脆弱性警告

- **概要**
MicrosoftのExchange Serverに存在するゼロデイ脆弱性は、Outlook Web Accessを標的にした攻撃で悪用されている。この脆弱性は特別に細工されたメールを通じて任意のJavaScriptコードを実行可能とし、Exchange Server 2016、2019およびSubscription Editionに影響を与える。パッチは未提供だが、Exchange Emergency Mitigation Serviceが自動緩和策を提供する。

- **URL**
https://www.bleepingcomputer.com/news/microsoft/microsoft-warns-of-exchange-zero-day-flaw-exploited-in-attacks/

- **備考**
この脆弱性の攻撃目的はリモートコード実行であり、攻撃者は細工メールを送信しOutlook Web AccessでのJavaScript実行を狙う。推奨対策はExchange Emergency Mitigation Serviceの有効化および最新のMitigation Tool適用である。類似のProxyLogonやProxyShell攻撃事例が過去に存在し、被害拡大の可能性が高い。将来的に正式パッチの適用が必要である。
