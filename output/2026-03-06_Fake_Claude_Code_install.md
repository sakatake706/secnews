- **タイトル**
Fake Claude Code install

- **概要**
攻撃者はInstallFix手法を用い、正規のCLIツールのインストール手順を装い悪意あるコマンドを実行させる手口を展開している。これにより利用者は不正なマルウェアをダウンロードし、情報窃取型のAmatera Stealerに感染する危険がある。攻撃は正規サイトのクローンを利用し巧妙に偽装されているため注意が必要である。

- **URL**
https://www.bleepingcomputer.com/news/security/fake-claude-code-install-guides-push-infostealers-in-installfix-attacks/

- **備考**
InstallFix攻撃は正規ツールのインストール手順を偽装し、[[Amatera Stealer]]を配布する。攻撃者は正規サイトのクローンを用い利用者を騙す。感染後は暗号資産ウォレットや認証情報が窃取される恐れがある。対策として公式サイトからの入手徹底、検索結果の広告回避、信頼できるダウンロード先の利用が推奨される。将来的に類似手法の拡大が懸念される。
