React Server Components DoS脆弱性

- **概要**
React Server Componentsに存在するCVE-2026-23864はサービス拒否を引き起こす脆弱性である。この脆弱性は特定のリクエスト処理時に過剰なリソース消費を招き、正常なサービス提供を妨害する可能性がある。攻撃者はこれを悪用しシステムの可用性を低下させることが可能である。
