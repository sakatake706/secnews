- **タイトル**
Entra Agent ID管理ロールの脆弱性

- **概要**
Microsoft EntraのAgent ID Administratorロールに範囲逸脱の脆弱性が存在し、攻撃者がテナント全体を乗っ取る可能性がある。この脆弱性はAIエージェント管理ロールがService Principalの奪取に利用される点に特徴がある。適切な権限管理と監査強化が求められる。

- **URL**
https://rocket-boys.co.jp/security-measures-lab/entra-id-agent-id-admin-role-leak-tenant-takeover/

- **備考**
Microsoft EntraのAgent ID Administratorロールの範囲逸脱により、攻撃者はService Principalを奪取しテナント全体の制御を取得可能である。攻撃は権限昇格を伴い、内部監査の不備が被害拡大を助長する。推奨対策は権限の最小化、ログ監視強化、AIエージェントのアクセス制御見直しである。将来的には類似の権限管理脆弱性に注意が必要である。攻撃者は高度な権限昇格技術を用いる可能性が高い。攻撃被害は企業のクラウド資産全体に及ぶ恐れがある。
