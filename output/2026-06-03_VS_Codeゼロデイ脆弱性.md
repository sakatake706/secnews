VS Codeゼロデイ脆弱性

- **概要**
Visual Studio Codeのゼロデイ脆弱性により攻撃者が悪意ある拡張機能をインストールしGitHub認証トークンを盗む手法が確認された。この脆弱性はVS Codeのサンドボックス化されたWebViewのメッセージ伝達システムを悪用し、ユーザーの操作を装ってトークンを取得するものである。現時点で公式パッチは未提供であり利用者はブラウザのクッキー削除などで対策可能である。

- **URL**
https://www.bleepingcomputer.com/news/security/vs-code-zero-day-lets-hackers-steal-github-tokens-in-one-click/

- **備考**
Visual Studio Codeのゼロデイ脆弱性はGitHubトークンの不正取得を可能にし攻撃者は悪意ある拡張機能を通じて権限を拡大する。攻撃はユーザーのクリック誘導を伴い被害拡大の恐れがある。対策としてブラウザのクッキー削除や拡張機能の警戒が推奨される。今後も類似のゼロデイ攻撃が増加する可能性が高い。
