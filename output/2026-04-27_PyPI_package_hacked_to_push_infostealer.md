- **タイトル**
PyPI package hacked to push infostealer

- **概要**
攻撃者はPython Package Indexのelementary-dataパッケージのワークフローの脆弱性を悪用し、悪意あるバージョン0.23.3を公開した。このバージョンはDockerイメージにも影響し、SSH鍵やクラウド認証情報、暗号通貨ウォレットなどの機密情報を盗む仕組みを含む。コミュニティメンバーの指摘で迅速に修正版が公開されたが、既にダウンロードしたユーザーは依然としてリスクが残る。

- **URL**
https://www.bleepingcomputer.com/news/security/pypi-package-with-11m-monthly-downloads-hacked-to-push-infostealer/

- **備考**
攻撃者はGitHub Actionsのスクリプトインジェクション脆弱性を利用し、正規のリリースパイプラインを悪用してマルウェア入りパッケージを配布した。これにより多様な秘密情報が漏洩する恐れがあるため、影響を受けたユーザーは全ての秘密情報のローテーションと安全な状態からの復元が推奨される。今後も類似の供給連鎖攻撃が増加する可能性が高い。
