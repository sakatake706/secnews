- **タイトル**
Critical React Native Metro脆弱性

- **概要**
React NativeのMetro開発サーバーに存在するCVE-2025-11953はOSコマンドインジェクションの脆弱性である。この脆弱性により認証なしで悪意あるコードが実行可能であり、WindowsおよびLinux環境で悪用されている。攻撃者はPowerShellベースのマルチステージローダーを用い、Microsoft Defenderを無効化してペイロードを実行している。攻撃は2025年12月から確認されている。

- **URL**
https://go.theregister.com/feed/www.theregister.com/2026/02/03/critical_react_native_metro_server/

- **備考**
攻撃者は認証回避とコマンド実行を狙い、PowerShellローダーで防御回避を行う。対策としては速やかなパッチ適用、ネットワーク監視強化、異常通信検知が重要である。今後も同種攻撃の拡大が懸念されるため継続的な監視が必要である。攻撃者[[不明]]の活動は活発化している。
