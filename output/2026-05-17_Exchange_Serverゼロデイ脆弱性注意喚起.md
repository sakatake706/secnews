Exchange Serverゼロデイ脆弱性注意喚起

- **概要**
米当局はMicrosoft Exchange Serverのゼロデイ脆弱性CVE-2026-42897について警告した。同脆弱性はOutlook Web Accessのウェブページ生成処理に存在し、クロスサイトスクリプティング攻撃を許す。実際の攻撃で悪用が確認されており、影響を受ける環境では迅速な対応が必要である。

- **URL**
https://www.security-next.com/184521

- **備考**
脆弱性はOutlook Web Accessの処理に起因し、攻撃者は任意のJavaScriptを実行可能である。CISAは行政機関に対策実施を求め、パッチ未提供のため使用中止も推奨される。将来的な被害拡大の恐れが高く、早急なセキュリティ対策が必要である。
