- **タイトル**
Microsoft Azure Monitor alerts abused

- **概要**
Microsoft Azure Monitorのアラートがコールバック型フィッシング攻撃に悪用されている。攻撃者はAzure Monitorの正規アラート機能を利用し、不正請求を装った警告メールを送信している。これにより受信者は偽の電話番号に連絡するよう誘導され、認証情報の窃取や不正送金などの被害が発生している。攻撃は正規のメール認証を通過し検知が困難であるため注意が必要である。

- **URL**
https://www.bleepingcomputer.com/news/security/microsoft-azure-monitor-alerts-abused-in-callback-phishing-campaigns/

- **備考**
Azure Monitorの正規機能を悪用した[[コールバック型フィッシング]]攻撃は、正規メール認証を通過し検知回避が可能である。攻撃者は請求関連のアラートを偽装し、被害拡大の恐れがある。対策としては不審な電話番号への連絡禁止、アラート内容の慎重な確認、二要素認証の導入が有効である。今後も類似手法の拡大が懸念されるため継続的な監視が必要である。
