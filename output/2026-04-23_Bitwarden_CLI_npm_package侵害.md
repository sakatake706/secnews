- **タイトル**
Bitwarden CLI npm package侵害

- **概要**
Bitwarden CLIのnpmパッケージが悪意あるコードにより一時的に侵害された。攻撃者は認証情報を盗むペイロードを含むバージョン2026.4.0を配布し、約1時間半の間に影響を及ぼした。調査ではエンドユーザーデータへのアクセスは確認されず、問題発覚後速やかに対応が実施された。

- **URL**
https://www.bleepingcomputer.com/news/security/bitwarden-cli-npm-package-compromised-to-steal-developer-credentials/

- **備考**
攻撃者はCI/CDパイプラインのGitHub Actionを悪用し、認証情報を収集するマルウェアを注入した。収集データは暗号化されGitHubの公開リポジトリに送信される。自己拡散機能も備え、被害拡大の恐れがあるため、影響を受けた開発者は全認証情報の変更と環境の精査が必須である。攻撃はTeamPCPに関連し、類似の供給網攻撃と連動している。攻撃の早期検知と対策強化が重要である。
