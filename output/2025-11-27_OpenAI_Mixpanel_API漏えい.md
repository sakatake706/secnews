- **タイトル**
OpenAI Mixpanel API漏えい

- **概要**
OpenAIの旧データ分析ベンダーMixpanelで発生した情報漏えいにより、API利用者のプロフィール情報が流出した。漏えいは11月9日に検知され、25日にOpenAIに報告された。影響はOpenAIプラットフォームのAPI利用者に限定され、ChatGPT一般利用者は対象外。OpenAIはMixpanelの利用を停止し、ベンダー全体のセキュリティ見直しを実施している。

- **URL**
https://go.theregister.com/feed/www.theregister.com/2025/11/27/openai_mixpanel_api/

- **備考**
OpenAIのAPI利用者情報漏えいはプラットフォーム限定の影響である。攻撃者の詳細は不明だが[[情報漏えい]]の可能性が高い。対策としてはベンダー管理強化、アクセス権限の見直し、監査体制の強化が必要である。将来的には類似事案の再発防止に向けた継続的な監視と迅速な対応体制構築が重要となる。
