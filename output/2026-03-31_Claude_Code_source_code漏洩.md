Claude Code source code漏洩

- **概要**
Anthropicが誤ってClaude CodeのソースコードをNPMパッケージに含めて公開した。公開されたファイルは約1900ファイル、50万行のコードを含み、内部機能の詳細が判明した。顧客データや認証情報の漏洩はなく、人的ミスによるものである。現在、DMCA通知で削除対応中である。
