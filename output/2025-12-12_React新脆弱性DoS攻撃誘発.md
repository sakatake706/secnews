- **タイトル**
React新脆弱性DoS攻撃誘発

- **概要**
React Server Componentsに新たな脆弱性が発見された。二つの高危険度のサービス拒否バグと一つのソースコード漏洩脆弱性が含まれる。これらは特定のHTTPリクエストによりサーバーを停止させたり、秘密情報を漏洩させる可能性がある。既存のReact2Shell脆弱性の修正が不完全であり、迅速なパッチ適用が必要である。

- **URL**
https://go.theregister.com/feed/www.theregister.com/2025/12/12/new_react_secretleak_bugs/

- **備考**
Reactの新脆弱性はサービス拒否と情報漏洩を引き起こす可能性が高い。攻撃者は特別に細工したHTTPリクエストを用いてサーバー機能を無限ループに陥れ、サービス停止を狙う。また、特定のサーバー関数の引数を悪用し、ソースコード内の秘密情報を抽出可能である。これらの脆弱性は既存のパッチで完全に対処されておらず、複数のバージョンで影響が継続している。攻撃者は北朝鮮や中国のグループとされ、被害拡大の恐れがあるため、迅速なアップデートと監視強化が推奨される。攻撃の手口は過去の大規模脆弱性と類似し、今後も注意が必要である。攻撃者[[北朝鮮]]、[[中国]]が関与。[[CVE-2025-55183]]、[[CVE-2025-55184]]、[[CVE-2025-67779]]が対象。
